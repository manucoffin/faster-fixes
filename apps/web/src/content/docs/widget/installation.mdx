---
title: Installation
description: Install the Faster Fixes widget in your React application.
---

```bash
npm install @fasterfixes/react
```

## Provider setup

The `FeedbackProvider` component must wrap your application:

```tsx
import { FeedbackProvider } from "@fasterfixes/react";

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html>
      <body>
        <FeedbackProvider apiKey={process.env.NEXT_PUBLIC_FF_API_KEY!}>
          {children}
        </FeedbackProvider>
      </body>
    </html>
  );
}
```

### Props

| Prop | Type | Required | Description |
|------|------|----------|-------------|
| `apiKey` | `string` | Yes | Your project API key from the dashboard |
| `apiOrigin` | `string` | No | API base URL. Defaults to `https://app.faster-fixes.com` |
| `classNames` | `object` | No | CSS class overrides for widget elements |
| `labels` | `object` | No | Custom text labels for the widget UI |

## Core package

For non-React frameworks, you can use the core client directly:

```bash
npm install @fasterfixes/core
```

```ts
import { FasterFixesClient } from "@fasterfixes/core";

const client = new FasterFixesClient({
  apiKey: "your-api-key",
  apiOrigin: "https://app.faster-fixes.com", // optional
});
```

The core package provides the API client, selector generation utilities, and browser info helpers without any React dependency.
