title: MCP Server
description: Use AI coding agents to fetch and resolve feedback without leaving your terminal.
---

The Faster Fixes MCP server lets AI coding agents interact with your feedback directly. Instead of opening the dashboard, your agent fetches new feedback, reads the context, locates the relevant code, and fixes the issue — all from your terminal.

## The workflow

1. Client leaves feedback via the widget
2. You ask your AI agent to check for new feedback
3. The agent calls the MCP server, gets structured feedback with full context
4. The agent reads the feedback, locates the code, and implements the fix
5. The agent marks the feedback as resolved

## Supported editors

- Claude Code
- Cursor
- VS Code (GitHub Copilot)
- Windsurf
- Codex
- Zed

## Requirements

- An agent token (created in Organization Settings → Agent Tokens)
- A project ID (found in Project Settings)

Get started with the [setup guide](/docs/mcp/setup).
