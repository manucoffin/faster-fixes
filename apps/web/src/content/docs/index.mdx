Client feedback during review cycles is chaotic — WhatsApp messages, vague screenshots, no page links. Faster Fixes replaces that with a website widget where clients just click and comment. All technical context (screenshot, URL, DOM selector, component tree, browser metadata) is captured transparently behind the scenes, so AI coding agents can fix issues directly.

## Key features

- **Website feedback widget** — Drop-in React component. Clients just click and comment — no training needed
- **Automatic context capture** — Screenshot, URL, DOM selector, component tree, browser metadata
- **Developer dashboard** — Feedback inbox with filtering and status management
- **MCP server** — AI agent integration for Claude Code, Cursor, VS Code, and others
- **GitHub integration** — Auto-create issues with bidirectional status sync

## How it works

1. Client clicks on your site and leaves feedback
2. AI agent fetches the feedback via the MCP server
3. Agent fixes the issue and marks the feedback as resolved

The dashboard gives you full visibility over every feedback item. You can step in at any point — review, reassign, or resolve tasks manually, just like any project management tool.

## Who it's for

Freelance developers, small web agencies, and teams building SaaS or web apps. Designed for React and Next.js stacks.
